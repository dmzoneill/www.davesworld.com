If you're not sure on how to get this code working do the following:
 
  • First highlight the code and then copy it.  Then paste the code into a new Notepad document or text editor.
  • Save the new file as a *.HTML document.
  • In your browser edit your bookmarks and add a new local bookmark which will link to the new hotmail.html file on your hard drive.
  • Open the file by clicking on your bookmarks and then selecting the new bookmark you have created for the file.
